    **startGame(difficulty: Difficulty):**
    1. `_uiState.update { it.copy(isLoading = true) }` — emit loading state
    2. `viewModelScope.launch { ... }` with `withContext(Dispatchers.Default)` around `generatePuzzle(difficulty)` — CPU-bound generation off main thread
    3. `undoStack.clear()` — clear previous game undo history
    4. Build fresh GameUiState from scratch (NOT `.copy()` of existing state — avoids carrying over errorCount, etc.):
       ```kotlin
       val givenMask = BooleanArray(81) { i -> puzzle.board[i] != 0 }
       _uiState.update {
           GameUiState(
               board = puzzle.board.copyOf(),
               solution = puzzle.solution.copyOf(),
               givenMask = givenMask,
               difficulty = difficulty,
               isLoading = false
           )
       }
       ```

    **selectCell(index: Int):**
    - Guard: `if (index !in 0..80) return`
    - `_uiState.update { it.copy(selectedCellIndex = index) }`
    - Given cells CAN be selected (per UI-SPEC interaction semantics: "selectCell(index) — but given cells are not selectable for INPUT"). Selection is allowed; only digit entry is blocked.

    **enterDigit(digit: Int):**
    - Guard: `if (digit !in 1..9) return`
    - `val state = _uiState.value`
    - `val idx = state.selectedCellIndex ?: return` — no selection = no-op
    - `if (state.givenMask[idx]) return` — given cells cannot accept input
    - Route based on `state.inputMode`:
      - `InputMode.FILL` -> call `applyFill(idx, digit, state)` (private method)
      - `InputMode.PENCIL` -> call `applyPencilMark(idx, digit, state)` (private method — implemented in Plan 03)
    - For Plan 02, implement `applyFill` fully. Implement `applyPencilMark` as a stub that does nothing (Plan 03 completes it).

    **applyFill(idx: Int, digit: Int, state: GameUiState):**
    1. Push undo action: `undoStack.addLast(GameAction.FillCell(idx, state.board[idx], state.pencilMarks[idx]))`
    2. `val newBoard = state.board.copyOf()` then `newBoard[idx] = digit`
    3. Check error: `val isError = digit != state.solution[idx]`
    4. `val newErrorCount = if (isError) state.errorCount + 1 else state.errorCount`
    5. Clear pencil marks in filled cell: `val newMarks = state.pencilMarks.copyOf().also { it[idx] = emptySet() }`
    6. Check completion: `val allCorrect = newBoard.indices.all { i -> newBoard[i] == state.solution[i] }`
    7. `_uiState.update { it.copy(board = newBoard, errorCount = newErrorCount, pencilMarks = newMarks, isComplete = allCorrect) }`
    8. If `allCorrect`: `viewModelScope.launch { _events.emit(GameEvent.Completed(newErrorCount)) }`

    **toggleInputMode():**
    ```kotlin
    _uiState.update { state ->
        state.copy(inputMode = when (state.inputMode) {
            InputMode.FILL -> InputMode.PENCIL
            InputMode.PENCIL -> InputMode.FILL
        })
    }
    ```
