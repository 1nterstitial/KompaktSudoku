<task type="auto">
  <name>Task 1: Generate keystore, configure signing, build signed APK</name>
  <files>app/build.gradle.kts, local.properties</files>
  <action>
1. Generate keystore with keytool (anonymous DN per locked decisions):
   ```
   keytool -genkeypair -v -keystore ~/kompakt-sudoku.jks -keyalg RSA -keysize 2048 -validity 10000 -alias release -dname "CN=KompaktSudoku, OU=Unknown, O=Unknown, L=Unknown, ST=Unknown, C=Unknown" -storepass [prompt user or generate] -keypass [same]
   ```
   IMPORTANT: Prompt the user for their desired keystore password before running keytool. Use the same password for both storepass and keypass for simplicity.

2. Add signing credentials to local.properties (already gitignored — confirmed):
   ```
   RELEASE_STORE_FILE=~/kompakt-sudoku.jks
   RELEASE_STORE_PASSWORD=[password]
   RELEASE_KEY_ALIAS=release
   RELEASE_KEY_PASSWORD=[password]
   ```

3. Add signingConfigs block to app/build.gradle.kts. Insert BEFORE the buildTypes block inside the android block:
   ```kotlin
   signingConfigs {
       create("release") {
           val props = java.util.Properties().apply {
               rootProject.file("local.properties").inputStream().use { load(it) }
           }
           storeFile = file(props["RELEASE_STORE_FILE"] as String)
           storePassword = props["RELEASE_STORE_PASSWORD"] as String
           keyAlias = props["RELEASE_KEY_ALIAS"] as String
           keyPassword = props["RELEASE_KEY_PASSWORD"] as String
       }
   }
   ```
   Then update the release buildType to reference it:
   ```kotlin
   release {
       isMinifyEnabled = false
       signingConfig = signingConfigs.getByName("release")
       proguardFiles(...)
   }
   ```

4. Run `./gradlew assembleRelease` to build the signed APK.

5. Verify the APK is signed: `jarsigner -verify app/build/outputs/apk/release/app-release.apk`
  </action>
  <verify>
    <automated>jarsigner -verify app/build/outputs/apk/release/app-release.apk 2>&1 | grep -q "jar verified" && echo "PASS: APK is signed" || echo "FAIL: APK not signed"</automated>
  </verify>
  <done>Signed release APK exists at app/build/outputs/apk/release/app-release.apk, jarsigner confirms it is verified.</done>
</task>
