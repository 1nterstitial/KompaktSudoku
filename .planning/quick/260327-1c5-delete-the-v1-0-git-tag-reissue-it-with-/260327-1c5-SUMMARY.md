## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] jarsigner reports APK as unsigned (false negative)**
- **Found during:** Task 1 verification
- **Issue:** `jarsigner -verify` reports "jar is unsigned" for APKs signed with APK v2 scheme. Modern Android APKs skip v1 (JAR) signing; jarsigner only checks v1.
- **Fix:** Used `apksigner verify --verbose` instead, which correctly reports "Verified using v2 scheme (APK Signature Scheme v2): true"
- **Files modified:** None — test method change only
- **Commit:** N/A (verification method change)

**2. [Rule 3 - Blocking] `java.util.Properties` unresolved in Kotlin DSL build script**
- **Found during:** Task 1 (first build attempt)
- **Issue:** `java.util.Properties()` is not auto-imported in Kotlin DSL `.kts` scripts; caused compile error "Unresolved reference: util"
- **Fix:** Added `import java.util.Properties` at top of `app/build.gradle.kts`
- **Files modified:** app/build.gradle.kts
- **Commit:** 534b937

**3. [Rule 3 - Blocking] ExpiredTargetSdkVersion lint error blocked assembleRelease**
- **Found during:** Task 1 (second build attempt)
- **Issue:** Android lint treats `targetSdk = 31` as a hard error ("Google Play requires API 33+"), failing the build
- **Fix:** Added `lint { disable += "ExpiredTargetSdkVersion" }` block with comment explaining this app targets Mudita Kompakt (not Play Store)
- **Files modified:** app/build.gradle.kts
- **Commit:** 534b937
