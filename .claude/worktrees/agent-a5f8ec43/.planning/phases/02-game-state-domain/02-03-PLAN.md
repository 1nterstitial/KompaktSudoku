    Replace the stub `applyPencilMark` with:
    ```kotlin
    private fun applyPencilMark(idx: Int, digit: Int, state: GameUiState) {
        val currentSet = state.pencilMarks[idx]
        val wasAdded = digit !in currentSet
        undoStack.addLast(GameAction.SetPencilMark(idx, digit, wasAdded))
        val newMarks = state.pencilMarks.copyOf()
        newMarks[idx] = if (wasAdded) currentSet + digit else currentSet - digit
        _uiState.update { it.copy(pencilMarks = newMarks) }
    }
    ```

    Key details:
    - `wasAdded = true` when the digit was NOT already present (it gets added) — undo must REMOVE it
    - `wasAdded = false` when the digit WAS present (it gets removed) — undo must ADD it back
    - `state.pencilMarks.copyOf()` creates a new array (shallow copy is safe since Set<Int> is immutable)
    - Toggle semantics: if digit in set, remove it; if not, add it

    **Modify GameViewModel.kt — implement undo():**

    Add public method:
    ```kotlin
    fun undo() {
        if (undoStack.isEmpty()) return
        when (val action = undoStack.removeLast()) {
            is GameAction.FillCell -> {
                _uiState.update { state ->
                    val newBoard = state.board.copyOf()
                    newBoard[action.cellIndex] = action.previousValue
                    val newMarks = state.pencilMarks.copyOf()
                    newMarks[action.cellIndex] = action.previousPencilMarks
                    // Recheck completion after undo (may no longer be complete)
                    val allCorrect = newBoard.indices.all { i -> newBoard[i] == state.solution[i] }
                    state.copy(
                        board = newBoard,
                        pencilMarks = newMarks,
                        isComplete = allCorrect
                    )
                }
            }
            is GameAction.SetPencilMark -> {
                _uiState.update { state ->
                    val newMarks = state.pencilMarks.copyOf()
                    newMarks[action.cellIndex] = if (action.wasAdded) {
                        state.pencilMarks[action.cellIndex] - action.digit
                    } else {
                        state.pencilMarks[action.cellIndex] + action.digit
                    }
                    state.copy(pencilMarks = newMarks)
                }
            }
        }
    }
    ```

    Key details:
    - `removeLast()` pops from end of ArrayDeque (LIFO stack behavior)
    - FillCell undo: restore `previousValue` (0 if was empty), restore `previousPencilMarks` (cleared on fill)
    - SetPencilMark undo: reverse the toggle (`wasAdded=true` -> remove, `wasAdded=false` -> add)
    - FillCell undo rechecks completion: if undoing the last correct fill, `isComplete` should go back to `false`
    - Do NOT decrement `errorCount` on undo — errors are permanent per SCORE-01 design (silent error tracking counts mistakes, not current board state)
